
# Boolean Algebra Laws 布尔代数定律

研究**逻辑运算**的代数性质。

| 定律名称    | 表达式                               |
| ------- | --------------------------------- |
| 幂等律     | A + A = A，A·A = A                 |
| 互补律     | A + A' = 1，A·A' = 0               |
| 单位元律    | A + 0 = A，A·1 = A                 |
| 零律      | A·0 = 0，A + 1 = 1                 |
| **交换律** | **A + B = B + A，A·B = B·A**       |
| **结合律** | **A + (B + C) = (A + B) + C 等**   |
| **分配律** | **A·(B + C) = A·B + A·C 等**       |
| 吸收律     | A + A·B = A，A·(A + B) = A         |
| **德摩根定律**   | **¬(A·B) = ¬A + ¬B，¬(A + B) = ¬A·¬B** |
