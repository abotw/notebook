5．设备中断处理程序通常须完成哪些工作？它对中断进行处理的过程包含哪些步骤？

【参考答案】设备中断处理程序要完成：①唤醒被阻塞的驱动进程；②保存被中断进程

的CPU现场信息；③转入相应的设备处理程序；④处理中断；⑤恢复被中断进程的CPU现场。

对中断进行处理的过程为：①测定是否有未响应的中断信号；②保护被中断进程的CPU现场；

③转入相应的设备处理程序；④处理中断；⑤恢复CPU现场并退出中断。
